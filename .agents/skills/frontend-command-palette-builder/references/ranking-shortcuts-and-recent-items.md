# Ranking, Shortcuts, and Recent Items

Predictability matters more than novelty.

Check:

- highest-ranked results reflect user intent, not just loose string matches
- shortcuts are visible where they matter and not duplicated on every row
- recent items feel helpful, not noisy or stale
- action results and navigation results are clearly distinguished
- grouped sections stay understandable when the query narrows results quickly

Prefer stable ranking rules over magical-feeling reshuffles.
