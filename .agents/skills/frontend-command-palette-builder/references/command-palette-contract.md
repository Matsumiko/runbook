# Command Palette Contract

Define the command palette as a product feature, not just a searchable modal.

Clarify:

- what opens it
- where it can be opened from
- whether it searches routes, records, actions, or all of them
- whether selection navigates, runs an action, opens another surface, or fills input
- whether recent items, suggestions, or pinned commands appear by default
- what permissions or unavailable states do to result visibility

The contract should explain why a command appears and what happens after selection.
