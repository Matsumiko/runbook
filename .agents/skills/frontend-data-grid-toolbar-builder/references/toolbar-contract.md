# Toolbar Contract

Define the toolbar before composing controls.

Capture:

- controlled surface: table, grid, directory, or search results
- primary controls: search, filters, sort, grouping, export, view toggle, or column visibility
- secondary controls and overflow candidates
- selection model and bulk-action rules
- result summary, active-filter chips, or saved-view cues
- state coverage: loading, zero selected, some selected, empty result, and error

Make the contract explicit about which actions are immediate and which are applied.
