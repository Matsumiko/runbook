# Metric Card Contract

Define each KPI card before styling it.

Capture:

- metric label and primary value
- value format: integer, currency, percent, duration, count, or compact notation
- time window or scope
- comparison baseline: previous period, target, benchmark, forecast, or none
- card actions: link, drill-down, filter, or passive summary only
- supporting elements: delta badge, sparkline, progress bar, badge, tooltip, or status note
- state coverage: loading, unavailable, empty, stale, and error

Do not let the visual treatment hide what the number actually means.
