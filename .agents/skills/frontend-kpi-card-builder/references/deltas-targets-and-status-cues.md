# Deltas, Targets, and Status Cues

Comparison cues must be disciplined.

Guidelines:

- always make the baseline explicit or omit the delta entirely
- positive and negative treatment should match product semantics, not generic green-is-good assumptions
- use arrows, badges, and semantic colors only when they improve comprehension
- targets and thresholds should read as context, not compete with the main value
- avoid mixing too many comparison signals in one card

If the card includes a sparkline, it should support the main number rather than replace clear comparison text.
