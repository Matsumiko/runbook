# Trigger And Anchor Contract

Context menus differ from normal dropdowns because trigger semantics change the user's expectation.

## Trigger paths

- Pointer context-click
- Keyboard invocation
- Long press or alternate mobile action

## Anchor behavior

- Cursor-anchored menus should appear near the invocation point.
- Object-anchored menus should still make the associated item obvious.
- Do not let the menu feel detached from the row, card, or object it controls.

## Scope questions

Clarify whether the menu applies to:

- one row or card
- a selected group
- a canvas object
- a file or folder node
