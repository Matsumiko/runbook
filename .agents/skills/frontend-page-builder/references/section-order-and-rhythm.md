# Section Order And Rhythm

Build pages with a clear top-to-bottom rhythm.

Prefer a structure like:

1. page title, hero, or route header
2. primary content or primary action area
3. supporting sections, detail blocks, or secondary actions
4. footer notes, related links, or lower-priority content

Keep rules like these in mind:

- do not let every section use the same visual weight
- keep vertical spacing consistent and intentional
- let headings and summaries create an obvious scan path
- keep side panels or tabs subordinate to the page's main job
- preserve the same pacing on mobile when sections stack
