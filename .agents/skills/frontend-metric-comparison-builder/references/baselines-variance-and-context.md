# Baselines, Variance, and Context

Variance needs explicit meaning.

Guidelines:

- always name the baseline or comparison partner
- make good, bad, and neutral movement follow product semantics, not generic color conventions
- keep units and decimal precision consistent across compared values
- do not let visual emphasis outrun interpretability
- highlight thresholds or benchmarks only when they improve decision support

If a metric has no trustworthy baseline, show that honestly instead of implying flat performance.
