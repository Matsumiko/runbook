# Comparison Surface Contract

Define the comparison surface before choosing the layout.

Capture:

- compared metric list
- baseline type: previous period, target, benchmark, cohort, plan, or peer
- comparison expression: absolute delta, percent delta, ratio, or variance band
- scope labels and time windows
- whether drill-down, notes, or explanations are needed
- state coverage: loading, no baseline, partial comparison, stale data, and error

Do not assume users know what a delta means without explicit context.
