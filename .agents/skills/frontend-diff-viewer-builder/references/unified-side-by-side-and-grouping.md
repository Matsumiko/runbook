# Unified, Side-By-Side, And Grouping

- Use unified mode when context flow matters more than strict side-by-side alignment.
- Use side-by-side mode when users need direct before-and-after comparison for each block.
- Group changes by section, field, or chunk when the artifact is dense.
- Keep line numbers, anchors, or field labels only when they help orientation.
- Do not rely on red-green coloring alone to explain changes.
