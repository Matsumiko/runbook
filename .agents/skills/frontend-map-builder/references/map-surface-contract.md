# Map Surface Contract

Use this reference to define the spatial task before choosing interactions.

## Common map jobs

- Store or asset locator
- Delivery or route preview
- Clustered search results
- Region or area overlay view

## Contract questions

Clarify:

- markers, routes, polygons, or mixed overlays
- whether side list or detail panel exists
- selection behavior between map and list
- clustering or aggregation needs
- what empty or unavailable states look like

## Pattern boundary

- Use chart-builder for non-spatial analytical visuals.
- Use page, search, or dashboard builders when the map is only one part of a broader route.
