# Chronology Grouping And Status Markers

Sequence meaning should be obvious before styling gets fancy.

## Grouping

- Group by date or milestone only when it improves scanability.
- Keep repeated low-signal events from overwhelming major moments.

## Status markers

- Use markers, chips, or icons to clarify state transitions.
- Do not rely on color alone for chronology meaning.
- Keep marker styles subordinate to the event content.

## Order clarity

- Make the reading direction obvious at a glance.
- If the order is non-standard, explain it in visible UI.
