# Axes Legends And Comparison Clarity

Visualization is only useful if the user can interpret it quickly.

## Axes

- Label units and scale honestly.
- Avoid decorative axes when they do not help interpretation.
- Keep tick density low enough to remain readable.

## Legends

- Show legends only when multiple series or categories require them.
- Keep legend labels plain and specific.
- If the chart can be understood directly from labels, avoid redundant legend chrome.

## Comparison clarity

- Do not use too many series at once.
- Use color consistently and sparingly.
- Make baseline or reference meaning explicit when comparisons matter.
