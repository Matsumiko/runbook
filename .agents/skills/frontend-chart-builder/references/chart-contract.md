# Chart Contract

Use this reference to define the question before choosing a visualization.

## Common chart jobs

- Trend over time
- Category comparison
- Composition or proportion
- Relationship or correlation
- Compact sparkline summary

## Contract questions

Clarify:

- what question the chart must answer
- which metric and units matter
- single series or multiple series
- time buckets or category grouping
- what empty or partial data looks like

## Pattern boundary

- Use dashboard-builder when the chart is only one panel in a broader analytics surface.
- Use map-builder when geography, location, or spatial context is central.
