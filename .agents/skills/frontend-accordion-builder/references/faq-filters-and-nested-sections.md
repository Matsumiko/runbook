# FAQ, Filters, and Nested Sections

Common disclosure jobs have different rules.

Use FAQ-style accordions when:

- users scan multiple questions before opening one
- the answer is the primary hidden content

Use filter or settings disclosures when:

- the summary row must show counts, selected values, or helper text
- multiple groups may remain open at once

Use nested disclosure only when:

- the hierarchy is real and improves navigation
- the user can still tell which level they are opening
