# Expansion Behavior and Density

Expansion should reduce cognitive load, not create layout chaos.

Check:

- dense stacks stay readable when one or several sections expand
- content rhythm stays stable between collapsed and expanded states
- motion reinforces orientation instead of calling attention to itself
- nested disclosure is used only when flatter structure would be harder to scan
- long-form content inside panels gets spacing and headings, not one raw text block

Prefer calm motion and predictable state retention.
