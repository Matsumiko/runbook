# Result Summary and Page Size

Pagination is not only arrows and numbers.

Check:

- result ranges are honest about what the user is seeing
- total counts are shown only when the backend actually provides them
- page-size selectors do not dominate the footer
- current-page context remains visible when summaries and controls share space
- empty filtered pages get recovery UX instead of a dead end

Keep count language calm, exact, and trustworthy.
