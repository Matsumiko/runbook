# Navigation And Section Grouping

Build settings pages around clear grouping.

Prefer a structure like:

1. settings navigation or section index
2. grouped sections with concise headings
3. section-level helper text where needed
4. isolated dangerous actions last

Keep rules like these in mind:

- group by user goal, not raw field schema
- keep sections easy to scan and return to
- do not mix destructive actions into routine preference blocks
- keep section rhythm consistent on desktop and mobile
