# Lane Card And Status Model

Kanban boards succeed when lane meaning and card density stay clear.

## Lanes

- Keep lane labels short and unambiguous.
- Show counts only when they help prioritization.
- Avoid visual weight that makes every lane compete equally if priority order exists.

## Cards

- Keep title and primary state visible first.
- Use chips, avatars, dates, or priorities only when they speed scanning.
- Avoid turning each card into a mini detail page.

## Status clarity

- Do not rely on color alone for state meaning.
- Keep blocked, overdue, or high-priority signals obvious but controlled.
