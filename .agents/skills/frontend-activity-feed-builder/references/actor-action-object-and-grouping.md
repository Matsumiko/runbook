# Actor Action Object And Grouping

Activity feeds need sentence-level clarity before styling.

## Event clarity

- Make actor, action, and object easy to read at a glance.
- Keep timestamps and metadata secondary to the main event statement.
- Use avatars or icons only when they help recognition.

## Grouping

- Group by date or event family only when it improves scanning.
- Avoid grouping that makes chronology harder to understand.

## Compaction

- Repeated low-signal events can be grouped or shortened.
- Major events should remain individually visible.
