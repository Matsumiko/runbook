# Task Bars Dependencies And Milestones

Planning surfaces must keep structure legible under density.

## Task bars

- Keep label and span readable together.
- Avoid bar styling that hides status meaning.

## Dependencies

- Show dependencies only when they matter to the planning task.
- Keep connectors understandable without visual clutter.

## Milestones

- Make milestone markers distinct from task spans.
- Use chips or badges only when they fit the surrounding product language.
