# Gantt Contract

Use this reference to define the planning model before building bars and labels.

## Common gantt jobs

- Project schedule
- Dependency planner
- Roadmap bar view
- Resource or workstream planner

## Contract questions

Clarify:

- what rows represent
- what scale and date range matter
- whether dependencies or milestones exist
- zoom levels or range switching
- row grouping or hierarchy
- empty-range and blocked states

## Pattern boundary

- Use timeline-builder for simple milestone chronology.
- Use calendar-builder for schedule grids or agenda views.
