# Event Density Navigation And Mobile Fallback

Calendar quality depends on scanability under real density.

## Event density

- Group, truncate, or summarize rather than letting the grid become illegible.
- Distinguish all-day, overlapping, and blocked periods clearly.

## Navigation

- Keep current period, previous/next actions, and jump-to-today behavior obvious.
- View switches should preserve orientation, not feel like separate apps.

## Mobile fallback

- Month grids and week planners often need agenda or list treatment on smaller screens.
- Preserve chronology and current-period context after collapsing the layout.
